# The Method — How This Plan Actually Runs

Most study plans fail the same way: weeks of tutorials, nothing shipped, no way to
prove anything was learned. This repo runs on a different loop.

## The Capability Loop

```
Mission → Definition of Done → Build → Defense Test → Publish → Asset
```

Every study day, cert, and project passes through it:

1. **Mission** — why this skill, tied to a real role or artifact. No learning for learning's sake.
2. **Definition of Done** — decided *before* opening any resource. Not "learn LangGraph" but "a working agent with a human-in-the-loop gate, committed to `labs/`."
3. **Build** — hands on keyboard. Watching and reading don't count as progress; only typed, running things do.
4. **Defense Test** — the quality gate: *can I explain this unaided, out loud, for two minutes?* If not, it doesn't ship — it goes back on the study list. AI-assisted building is fine; AI-dependent shipping is not.
5. **Publish** — the lab, the write-up, the diagram. Public by default.
6. **Asset** — every project ends as something reusable: an interview story, a tool, an article.

## Retention mechanics (the evidence-based part)

- **Recall close:** every session ends with 15 minutes writing what was learned *from memory* — retrieval practice beats re-reading (Rowland 2014, 159 experiments).
- **Feynman check:** each major concept gets explained on a blank page in plain language; the stuck points are the study list (self-explanation effect, Bisra et al., g ≈ 0.55).
- **Weekly mini-exam:** 5–10 self-written questions, answered cold. Misses go on a gap list that gets cleared before new material.
- **Streak:** [PROGRESS.md](./PROGRESS.md) is the chain of X's.

## Certifications are side effects

Exams get booked *before* studying (deadline pressure is a feature), but the prep is
building the lab, not grinding dumps. The cert notarizes the capability; it doesn't
substitute for it.

## One rule above all

**Publish what you already know before learning the next thing.** Evidence compounds;
intentions don't.
