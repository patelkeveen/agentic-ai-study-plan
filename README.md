# 🚀 60-Day AI Security & Automation — Build in Public

> **Day 0 of 60** | Started: _July 15, 2026_ | Built by [Keveen Patel](https://github.com/patelkeveen)

I'm a Senior IT Engineer specializing in Microsoft 365, Entra ID, and cloud security. I'm spending 60 days (5 hours/day, 300 hours total) building production-grade AI automation and security skills — **in public**.

**Why?** Because [non-human identities outnumber humans 144:1](https://labs.cloudsecurityalliance.org/research/csa-whitepaper-nonhuman-identity-agentic-ai-governance-v1-cs/) in cloud environments, and **48% of cybersecurity professionals name agentic AI as the #1 attack vector for 2026**. The intersection of AI automation and identity security is the most valuable unclaimed niche in the industry right now.
